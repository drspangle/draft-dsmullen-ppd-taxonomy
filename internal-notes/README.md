# Internal Notes

These notes are local to this repository and are intentionally kept separate
from the rendered Internet-Draft contents.

Use this directory to capture:

- taxonomy-draft editorial history
- open modeling questions that are not yet ready to become draft text
- local and external source-material pointers that inform taxonomy scope
- working direction for the core-vocabulary and extension model

Do not put settled taxonomy semantics here if they belong in the draft itself.

## Files

- `draft-work-log.md`: running log of taxonomy-draft changes, criticism, and
  design decisions
- `source-material.md`: local PPD repository inputs that should shape the
  taxonomy
- `external-taxonomy-sources.md`: open external standards and research sources
  that may inform the PPD taxonomy model
- `core-vocabulary-direction.md`: working direction for the baseline core
  vocabulary, extension model, and source-selection rules
- `cross-draft-coordination.md`: current coordination findings and sequencing
  across the taxonomy, protocol, and architecture drafts
